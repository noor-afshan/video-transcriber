# Roadmap

Future enhancements for the Transcribe project.

## Documentation Enhancements

- [ ] **MkDocs Material static site** - Professional documentation site with search, dark mode, and better navigation
- [ ] **API documentation generation** - Auto-generate docs from Python docstrings using mkdocstrings
- [ ] **Video/GIF demos** - Visual walkthroughs of key workflows

## Project Infrastructure

- [ ] **GitHub Actions for linting** - Automated code quality checks (Black, Ruff, mypy)
- [ ] **pyproject.toml** - Enable `pip install .` for easier setup
- [ ] **Unit tests** - Test coverage for core modules
- [ ] **GitHub Actions for tests** - CI pipeline for automated testing

## Feature Ideas

- [ ] **Cross-platform support** - Linux/macOS compatibility
- [ ] **NVIDIA GPU support** - CUDA backend option alongside Intel Arc
- [ ] **Real-time transcription** - Live transcription during recording
- [ ] **Transcript export formats** - SRT, VTT, JSON output options
- [ ] **Web UI** - Browser-based interface for transcription
